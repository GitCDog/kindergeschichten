=== NÄCHSTE SCHRITTE ZUM STARTEN ===

1. ABHÄNGIGKEITEN INSTALLIEREN
   cd /pfad/zu/kindergeschichten-automation
   python -m venv venv
   venv\Scripts\activate
   pip install -r orchestrator/requirements.txt

2. API-KEYS EINTRAGEN (Optional für Tests)
   Öffne orchestrator\.env und füge ein:
   ELEVENLABS_API_KEY=your_api_key  # Für Audio-Generierung erforderlich
   ANTHROPIC_API_KEY=your_key       # Optional (Template-Generator funktioniert offline)

3. HINTERGRUNDBILDER HINZUFÜGEN
   Kopiere deine Bilder in:
   orchestrator\images\

4. STARTEN
   cd orchestrator
   python main.py

=== KONFIGURATION ===
- config.yaml: Alle Einstellungen (Story-Typ, Voice, Upload-Zeit, etc.)
- Template-Generator: Funktioniert OFFLINE ohne API-Keys
- Claude Generator: Optional für kreativere Stories (benötigt API-Key)
